Conexión PHP con MySQL

Guía paso a paso para conectar tu aplicación PHP con MySQL

¿Para qué sirve esta conexión?

Conectar PHP con MySQL es una habilidad fundamental para crear aplicaciones web dinámicas. Te permite almacenar, leer y manipular datos de una base de datos MySQL, lo que es esencial para crear sitios interactivos, como foros, tiendas en línea y más.

Requisitos

  • PHP: Necesitas tener PHP instalado en tu máquina.
  • MySQL: Deberás tener un servidor MySQL corriendo localmente o en un servidor remoto.
  • PowerShell: Usaremos PowerShell para ejecutar comandos en este tutorial.

Instalación Paso a Paso

A continuación te explicamos los pasos necesarios para instalar y configurar PHP y MySQL en tu máquina, y conectar ambos con un simple script PHP.

1. Instalar XAMPP o WAMP

Para facilitar la instalación, puedes usar herramientas como XAMPP o WAMP, que incluyen Apache, PHP y MySQL.

2. Descargar e Instalar PHP y MySQL (si no usas XAMPP o WAMP)

Si prefieres instalar PHP y MySQL manualmente, sigue estos pasos:

Para instalar PHP:

Descarga la última versión de PHP desde aquí y extrae el archivo en una carpeta, por ejemplo, C:\php.

Para agregar PHP a las variables de entorno:

 [System.Environment]::SetEnvironmentVariable('Path', $env:Path + ";C:\php", [System.EnvironmentVariableTarget]::Machine)

Para instalar MySQL:

Descarga MySQL desde aquí e instálalo.

3. Iniciar los Servicios

Para iniciar MySQL (si no usas XAMPP):

 mysqld

Para iniciar PHP (si no usas XAMPP):

 php -S localhost:8000

4. Conectar PHP con MySQL

Ahora que tienes PHP y MySQL funcionando, el siguiente paso es conectar ambos. Aquí tienes un ejemplo básico de cómo hacerlo:

<?php
$servidor = "localhost";
$usuario = "root";
$contraseña = "";
$baseDeDatos = "mi_base_de_datos";

$conn = new mysqli($servidor, $usuario, $contraseña, $baseDeDatos);

if ($conn->connect_error) {
    die("Conexión fallida: " . $conn->connect_error);
} else {
    echo "Conexión exitosa!";
}
?>

5. Verificar la Conexión

Guarda tu archivo PHP (por ejemplo, conexion.php) en la carpeta de tu servidor web y accede a él en tu navegador para verificar la conexión.
