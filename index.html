Conexión PHP con MySQL

Guía paso a paso para conectar tu aplicación PHP con MySQL

¿Para qué sirve esta conexión?

Conectar PHP con MySQL es una habilidad fundamental para crear aplicaciones web dinámicas. Te permite almacenar, leer y manipular datos de una base de datos MySQL, lo que es esencial para crear sitios interactivos, como foros, tiendas en línea y más.

Requisitos

  • PHP: Necesitas tener PHP instalado en tu máquina.
  • MySQL: Deberás tener un servidor MySQL corriendo localmente o en un servidor remoto.
  • PowerShell: Usaremos PowerShell para ejecutar comandos en este tutorial.

Instalación Paso a Paso

Verificar PHP y MySQL

php -v

Verifica que PHP esté correctamente instalado.

mysql --version

Verifica que MySQL esté funcionando correctamente.

Habilitar MySQLi o PDO

Asegúrate de que la extensión MySQLi o PDO esté habilitada en tu archivo `php.ini`.

Verificar MySQL Activo

netstat -an | find "3306"

Verifica que el puerto 3306 esté activo para conexiones de MySQL.

Conectar PHP con MySQL

Aquí tienes el código básico para establecer la conexión:

<?php
$servidor = "localhost";
$usuario = "root";
$contraseña = "";
$baseDeDatos = "mi_base_de_datos";
$conn = new mysqli($servidor, $usuario, $contraseña, $baseDeDatos);
if ($conn->connect_error) {
    die("Conexión fallida: " . $conn->connect_error);
} else {
    echo "Conexión exitosa!";
}
?>
